## API Contract Preserved
- POST {contextPath}/id/_generate
  - Request: { RequestInfo, idRequests: [{idName, tenantId, format?, count?}] }
  - Response: { responseInfo, idResponses: [{id}] }
  - Errors: { ResponseInfo, Errors: [{code, message}] }

## Key Decisions
- MDMS fetches format + city code in a single HTTP call (optimization over Java's separate calls)
- Sequence names validated as alphanumeric+underscore only (SQL injection prevention)
- Financial year logic matches Java: April-March cycle
- IST timezone mapped to Asia/Kolkata (IANA)
- `lib/pq` v1.12.3 for PostgreSQL driver
